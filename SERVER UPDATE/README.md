# oRent — Fleet Management System

A dark-themed, Mercedes-Benz inspired car rental management system built with **PHP 8 + MySQL**.

## Quick Start

👉 See **[SETUP.md](SETUP.md)** for full installation instructions.

**TL;DR:**
1. Install [XAMPP](https://www.apachefriends.org)
2. Copy this folder to `C:\xampp\htdocs\oRentPHP\`
3. Import `database.sql` into a MySQL database named `orent`
4. Open `http://localhost/oRentPHP/`

## Tech Stack

- **Backend:** PHP 8+ with PDO (no frameworks)
- **Database:** MySQL
- **Frontend:** Tailwind CSS (CDN), jQuery, Select2
- **Server:** Apache via XAMPP (local) or any PHP host
